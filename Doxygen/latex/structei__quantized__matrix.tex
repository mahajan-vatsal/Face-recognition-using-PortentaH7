\doxysubsection{Detailed Description}
Another matrix structure that allocates a matrix on the {\bfseries{heap}}. Freeing happens by calling {\ttfamily delete} on the object or letting the object go out of scope. We use this for the filterbanks, as we quantize these operations to save memory.
