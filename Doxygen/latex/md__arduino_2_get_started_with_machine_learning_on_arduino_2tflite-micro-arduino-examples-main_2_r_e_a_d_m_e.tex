The officially supported Tensor\+Flow Lite Micro library for Arduino resides in the \href{https://github.com/tensorflow/tflite-micro-arduino-examples}{\texttt{ tflite-\/micro-\/arduino-\/examples}} Git\+Hub repository. To install the in-\/development version of this library, you can use the latest version directly from the Git\+Hub repository. This requires you clone the repo into the folder that holds libraries for the Arduino IDE. The location for this folder varies by operating system, but typically it\textquotesingle{}s in {\ttfamily \texorpdfstring{$\sim$}{\string~}/\+Arduino/libraries} on Linux, {\ttfamily \texorpdfstring{$\sim$}{\string~}/\+Documents/\+Arduino/libraries/} on Mac\+OS, and {\ttfamily My Documents\textbackslash{}Arduino\textbackslash{}Libraries} on Windows.

Once you\textquotesingle{}re in that folder in the terminal, you can then grab the code using the git command line tool\+:


\begin{DoxyCode}{0}

\end{DoxyCode}

\DoxyCodeLine{git\ pull}
