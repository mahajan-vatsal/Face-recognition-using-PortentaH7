\doxysubsection{Detailed Description}
Holds the output of visual anomaly detection (FOMO-\/\+AD) 

If visual anomaly detection is enabled (e.\+g. {\ttfamily EI\+\_\+\+CLASSIFIER\+\_\+\+HAS\+\_\+\+VISUAL\+\_\+\+ANOMALY == 1}), then the output results will be a pointer to an array of grid cells of size {\ttfamily visual\+\_\+ad\+\_\+count}, as given by \href{https://docs.edgeimpulse.com/reference/ei_impulse_result_bounding_box_t}{\texttt{ ei\+\_\+impulse\+\_\+result\+\_\+bounding\+\_\+box\+\_\+t}}.

The visual anomaly detection result is stored in {\ttfamily visual\+\_\+ad\+\_\+result}, which contains the mean and max values of the grid cells.
