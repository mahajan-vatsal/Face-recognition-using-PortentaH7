\doxysubsection{Detailed Description}
Holds information for a single bounding box. 

If object detection is enabled (i.\+e. {\ttfamily EI\+\_\+\+CLASSIFIER\+\_\+\+OBJECT\+\_\+\+DETECTION == 1}), then inference results will be one or more bounding boxes. The bounding boxes with the highest confidence scores (assuming those scores are equal to or greater than {\ttfamily EI\+\_\+\+CLASSIFIER\+\_\+\+OBJECT\+\_\+\+DETECTION\+\_\+\+THRESHOLD}), given by the {\ttfamily value} member, are returned from inference. The total number of bounding boxes returned will be at least {\ttfamily EI\+\_\+\+CLASSIFIER\+\_\+\+OBJECT\+\_\+\+DETECTION\+\_\+\+COUNT}. The exact number of bounding boxes is stored in {\ttfamily bounding\+\_\+boxes\+\_\+count} field of \mbox{[}\doxylink{structei__impulse__result__t}{ei\+\_\+impulse\+\_\+result\+\_\+t}\mbox{]}/\+C++ Inference SDK Library/structs/ei\+\_\+impulse\+\_\+result\+\_\+t.\+md).

A bounding box is a rectangle that ideally surrounds the identified object. The ({\ttfamily x}, {\ttfamily y}) coordinates in the struct identify the top-\/left corner of the box. {\ttfamily label} is the predicted class with the highest confidence score. {\ttfamily value} is the confidence score between \mbox{[}0.\+0..1.\+0\mbox{]} of the given {\ttfamily label}.
