\doxysubsection{Detailed Description}
Abstract interface that returns Tf\+Lite\+Registrations given op codes or custom op names. This is the mechanism that ops being referenced in the flatbuffer model are mapped to executable function pointers (Tf\+Lite\+Registrations).
