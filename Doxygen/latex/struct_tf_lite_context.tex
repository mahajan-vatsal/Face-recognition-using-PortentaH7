\doxysubsection{Detailed Description}
{\ttfamily \doxylink{struct_tf_lite_context}{Tf\+Lite\+Context}} allows an op to access the tensors.

{\ttfamily \doxylink{struct_tf_lite_context}{Tf\+Lite\+Context}} is a struct that is created by the TF Lite runtime and passed to the "{}methods"{} (C function pointers) in the {\ttfamily \doxylink{struct_tf_lite_registration}{Tf\+Lite\+Registration}} struct that are used to define custom ops and custom delegate kernels. It contains information and methods (C function pointers) that can be called by the code implementing a custom op or a custom delegate kernel. These methods provide access to the context in which that custom op or custom delegate kernel occurs, such as access to the input and output tensors for that op, as well as methods for allocating memory buffers and intermediate tensors, etc.

See also {\ttfamily Tf\+Lite\+Opaque\+Context}, which is an more ABI-\/stable equivalent.
