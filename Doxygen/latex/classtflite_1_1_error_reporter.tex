\doxysubsection{Detailed Description}
A functor that reports error to supporting system. Invoked similar to printf.

Usage\+: \doxylink{classtflite_1_1_error_reporter}{Error\+Reporter} foo; foo.\+Report("{}test \%d"{}, 5); or va\+\_\+list args; foo.\+Report("{}test \%d"{}, args); // where args is va\+\_\+list

Subclass \doxylink{classtflite_1_1_error_reporter}{Error\+Reporter} to provide another reporting destination. For example, if you have a GUI program, you might redirect to a buffer that drives a GUI error log box.
