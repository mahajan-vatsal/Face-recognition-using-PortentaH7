\doxysubsection{Detailed Description}
Holds the output of inference, anomaly results, and timing information. 

{\ttfamily \doxylink{structei__impulse__result__t}{ei\+\_\+impulse\+\_\+result\+\_\+t}} holds the output of {\ttfamily run\+\_\+classifier()}. If object detection is enabled, then the output results is a pointer to an array of bounding boxes of size {\ttfamily bounding\+\_\+boxes\+\_\+count}, as given by \href{https://docs.edgeimpulse.com/reference/ei_impulse_result_bounding_box_t}{\texttt{ ei\+\_\+impulse\+\_\+result\+\_\+bounding\+\_\+box\+\_\+t}}. Otherwise, results are stored as an array of classification scores, as given by \href{https://docs.edgeimpulse.com/reference/ei_impulse_result_classification_t}{\texttt{ ei\+\_\+impulse\+\_\+result\+\_\+classification\+\_\+t}}.

If anomaly detection is enabled (e.\+g. {\ttfamily EI\+\_\+\+CLASSIFIER\+\_\+\+HAS\+\_\+\+ANOMALY == 1}), then the anomaly score will be stored as a floating point value in {\ttfamily anomaly}.

Timing information is stored in an \href{https://docs.edgeimpulse.com/reference/ei_impulse_result_timing_t}{\texttt{ ei\+\_\+impulse\+\_\+result\+\_\+timing\+\_\+t}} struct.
