\doxysubsection{Detailed Description}
A union of pointers that points to memory for a given tensor.

Do not access these members directly, if possible, use {\ttfamily Get\+Tensor\+Data\texorpdfstring{$<$}{<}TYPE\texorpdfstring{$>$}{>}(tensor)} instead, otherwise only access {\ttfamily .data}, as other members are deprecated.
