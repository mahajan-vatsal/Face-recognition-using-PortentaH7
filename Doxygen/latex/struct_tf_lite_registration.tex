\doxysubsection{Detailed Description}
{\ttfamily \doxylink{struct_tf_lite_registration}{Tf\+Lite\+Registration}} defines the implementation of an operation (a built-\/in op, custom op, or custom delegate kernel).

It is a struct containing "{}methods"{} (C function pointers) that will be invoked by the TF Lite runtime to evaluate instances of the operation.

See also {\ttfamily Tf\+Lite\+Registration\+External} which is a more ABI-\/stable equivalent.
