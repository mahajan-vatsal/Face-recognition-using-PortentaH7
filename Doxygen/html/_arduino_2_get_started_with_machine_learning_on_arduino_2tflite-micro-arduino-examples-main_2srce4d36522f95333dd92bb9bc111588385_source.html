Machine Vision using Portenta H7 2
This project aims to develop a face recognition-based access control system using the Arduino Portenta H7 and Vision Shield, leveraging Edge Impulse for machine learning. The system captures facial images, processes them locally using an AI model deployed on the Portenta H7 and determines access based on authorised personnel.

28// Helper class to perform a simulated kernel (i.e. TfLiteRegistration)
29// lifecycle (init, prepare, invoke). All internal allocations are handled by
30// this class. Simply pass in the registration, list of required tensors, inputs
31// array, outputs array, and any pre-builtin data. Calling Invoke() will
32// automatically walk the kernel and outputs will be ready on the TfLiteTensor
33// output provided during construction.
34class KernelRunner {
35 public:
36 KernelRunner(const TfLiteRegistration& registration, TfLiteTensor* tensors,
37 int tensors_size, TfLiteIntArray* inputs,
38 TfLiteIntArray* outputs, void* builtin_data,
39 TfLiteIntArray* intermediates = nullptr);
40
41 // Calls init and prepare on the kernel (i.e. TfLiteRegistration) struct. Any
42 // exceptions will be DebugLog'd and returned as a status code.
43 TfLiteStatus InitAndPrepare(const char* init_data = nullptr,
44 size_t length = 0);
45
46 // Calls init, prepare, and invoke on a given TfLiteRegistration pointer.
47 // After successful invoke, results will be available in the output tensor as
48 // passed into the constructor of this class.
49 TfLiteStatus Invoke();
50
51 // Calls Free on a given TfLiteRegistration pointer(if it's implemented).
52 // After successful Free, kTfLiteOk status will be returned. If Free is not
53 // implemented for a given kernel kTfLiteError will be returned.
54 TfLiteStatus Free();
55
56 // Returns a pointer to the internal MockMicroGraph which KernelRunner uses
57 // to stub out MicroGraph methods and track invocations on each subgraph.
58 MockMicroGraph* GetMockGraph() { return &mock_micro_graph_; }
59
60 // Returns true if all temp buffer in tests are deallocated.
61 // TODO(b/209453859): move this function to private after deallocation checks
62 // are enabled for all kernel tests.
63 bool ValidateTempBufferDeallocated();

65 private:

69 TfLiteContext context_ = {};
70 TfLiteNode node_ = {};
71 const TfLiteRegistration& registration_;
72
73 SingleArenaBufferAllocator* allocator_;
74 MockMicroGraph mock_micro_graph_;
75 FakeMicroContext fake_micro_context_;
76};
