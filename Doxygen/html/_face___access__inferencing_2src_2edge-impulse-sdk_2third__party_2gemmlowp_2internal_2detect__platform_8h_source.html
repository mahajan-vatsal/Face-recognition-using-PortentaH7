Machine Vision using Portenta H7 2
This project aims to develop a face recognition-based access control system using the Arduino Portenta H7 and Vision Shield, leveraging Edge Impulse for machine learning. The system captures facial images, processes them locally using an AI model deployed on the Portenta H7 and determines access based on authorised personnel.

21// Our inline assembly path assume GCC/Clang syntax.
22// Native Client doesn't seem to support inline assembly(?).

24#define GEMMLOWP_ALLOW_INLINE_ASM
25#endif

93// Detect MIPS MSA.

95// TODO: Perhaps, eventually support MSA optimizations on big-endian CPUs?
96#if defined(GEMMLOWP_MIPS) && (__mips_isa_rev >= 5) && defined(__mips_msa) && \

99#endif

104#endif

106#if defined(GEMMLOWP_MSA) && defined(GEMMLOWP_MIPS_64)

110// compiler define for AVX2 -D GEMMLOWP_ENABLE_AVX2
111// Detect AVX2

113#define GEMMLOWP_AVX2
