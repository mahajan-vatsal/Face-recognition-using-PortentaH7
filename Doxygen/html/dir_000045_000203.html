Machine Vision using Portenta H7 2
This project develops a QR code reader using the ESP32 microcontroller and the FOMO algorithm on Edge Impulse, achieving 75.9% F1 score and 60% accuracy. The trained model, optimized with TensorFlow Lite, classifies QR codes into three categories: Achal, Rudhresh, and Wings.
