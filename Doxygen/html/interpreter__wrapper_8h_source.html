Machine Vision using Portenta H7 2
This project aims to develop a face recognition-based access control system using the Arduino Portenta H7 and Vision Shield, leveraging Edge Impulse for machine learning. The system captures facial images, processes them locally using an AI model deployed on the Portenta H7 and determines access based on authorised personnel.

27// Allocation Recording is mutually exclusive from the PreserveAllTensors
28// debugging feature because PreserveAllTensors uses the LinearMemoryPlanner.
29// This means that the Allocations recorded by the RecordingMicroAllocator
30// wouldn't be accurate because the GreedyMemoryPlanner would have to be used.
31// So this Enum was made to represent the two possible modes/configs you can use
32// the python interpreter for.
33enum InterpreterConfig {
34 kAllocationRecording = 0,
35 kPreserveAllTensors = 1,
36};
