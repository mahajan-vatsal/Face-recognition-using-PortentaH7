Machine Vision using Portenta H7 2
This project aims to develop a face recognition-based access control system using the Arduino Portenta H7 and Vision Shield, leveraging Edge Impulse for machine learning. The system captures facial images, processes them locally using an AI model deployed on the Portenta H7 and determines access based on authorised personnel.

67/*
68 * kiss_fft_alloc
69 *
70 * Initialize a FFT (or IFFT) algorithm's cfg/state buffer.
71 *
72 * typical usage: kiss_fft_cfg mycfg=kiss_fft_alloc(1024,0,NULL,NULL);
73 *
74 * The return value from fft_alloc is a cfg buffer used internally
75 * by the fft routine or NULL.
76 *
77 * If lenmem is NULL, then kiss_fft_alloc will allocate a cfg buffer using malloc.
78 * The returned value should be free()d when done to avoid memory leaks.
79 *
80 * The state can be placed in a user supplied buffer 'mem':
81 * If lenmem is not NULL and mem is not NULL and *lenmem is large enough,
82 * then the function places the cfg in mem and the size used in *lenmem
83 * and returns mem.
84 *
85 * If lenmem is not NULL and ( mem is NULL or *lenmem is not large enough),
86 * then the function returns NULL and places the minimum cfg
87 * buffer size in *lenmem.
88 * */
89
90kiss_fft_cfg kiss_fft_alloc(int nfft,int inverse_fft,void * mem,size_t * lenmem,size_t * memallocated = NULL);
